# Bhavani Hospital - Website Redesign

![Bhavani Hospital Logo](public/Logo_transparent.png)

A modern, responsive, and highly interactive Single Page Application (SPA) built for Bhavani Hospital. This project completely redesigns the hospital's digital presence with premium UI/UX, prioritizing patient experience, clean aesthetics, and immediate access to critical healthcare information.

## 🚀 Technologies Used

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v4)
- **Icons:** React Icons
- **Animations:** Framer Motion
- **Carousels/Sliders:** Swiper.js
- **Image Viewing:** Yet-Another-React-Lightbox
- **Routing:** React Router v6

## ✨ Key Features

- **Dynamic Modern UI:** Glassmorphism effects, smooth micro-animations, and hover states using Framer Motion.
- **Fully Responsive:** Optimized layouts for mobile, tablet, and desktop devices.
- **Interactive Gallery:** Filterable hospital gallery with a built-in lightbox viewer.
- **Doctor Profiles:** Professional cards displaying doctor specializations, qualifications, and consultation details.
- **Services & Facilities:** Clear, visually appealing grids showcasing the hospital's capabilities.
- **Integrated Maps:** Embedded Google Maps in the Contact section for easy navigation.
- **Performance Optimized:** Fast build times and hot-module replacement via Vite.

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yashkumarinnovatehive-arch/Bhavani_Hospital.git
   cd "Bhavani Hospital Pune"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   *The application will be available at `http://localhost:5173`*

4. **Build for production:**
   ```bash
   npm run build
   ```
